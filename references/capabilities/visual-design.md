# Visual Design Capability

Use visual design guidance for new screens, brand-sensitive pages, redesigns, dashboards, games, or tools where visual polish affects usability.

## Fallback Guidance

Cover:

- Visual hierarchy.
- Layout density.
- Color and contrast.
- Typography scale.
- Icon and control choices.
- Motion or transition needs.
- Asset needs.

Prefer existing design systems and local components over new visual languages.
