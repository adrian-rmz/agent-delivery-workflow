# Research Capability

Use research only when it affects product, technical, legal, pricing, competitor, API, or market decisions.

## Routing

- Use installed structured research skills when the task needs repeatable extraction, many sources, or API-backed data.
- Use normal browser or web search for lightweight competitor, pricing, documentation, or current-fact checks.
- Use official sources for technical APIs and high-stakes facts.
- If no external research skill is installed, recommend one only when it would materially improve the result.

## Artifact

Save `_delivery/research/<slug>.md` only when research was performed.

Include:

- Questions researched.
- Sources consulted.
- Findings.
- Implications for the idea/spec/plan.
- Uncertainties.
