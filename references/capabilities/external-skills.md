# External Skills Capability

Maintain `_delivery/context/skills.md` as the routing map for installed and recommended capabilities.

## Rules

- If an external skill is installed and relevant, use it.
- If it is missing but valuable, recommend it and continue with internal fallback instructions.
- Do not install skills automatically.
- Do not fail a workflow only because a recommended skill is missing.

## Recommended Categories

- Research: structured market or competitor extraction when worth the setup.
- Web UI: frontend design, web accessibility, React, Next.js, or framework-specific guidance.
- Mobile: Expo, React Native, iOS, or Android design guidance.
- Data: database, analytics, spreadsheet, or BI-specific skills.

## `_delivery/context/skills.md` Sections

- Installed skills.
- Recommended skills.
- When to use each skill.
- Fallback behavior.
